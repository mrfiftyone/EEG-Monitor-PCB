# EEG Monitor Prototype

An open-source brainwave monitor prototype built with KiCad.

## Hardware Overview
* **Microcontroller:** ESP32-WROOM-32E (Wi-Fi/Bluetooth enabled)
* **ADC:** Texas Instruments ADS1299 (Low-Noise, 8-channel, 24-bit analog-to-digital converter)

## Manufacturing
The Gerber files required for PCB fabrication (copper, silkscreen, solder mask, edge cuts, and drill files) are bundled in the release section or located in the `EEG_Gerbers` folder.

## How to Open
To view or modify this project, you will need [KiCad](https://www.kicad.org/). Open the `EEG-Monitor.kicad_pro` file in the KiCad project manager.
